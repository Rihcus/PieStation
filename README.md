# PieStation
Raspberry Pi software for remote controlling your house. IR, 434mhz, audio, video, beacons and more! Domotica for the win!

# Hardware used
- Raspberry Pi 1
- IR Led
- Breadboard
- IR Receiver (TSOP2236)
- 434mhz module (TX433)

# Software used
- NodeJS
- lirc
- Raspbian
